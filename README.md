# Building resume with ReactJS and Webpack

Create a new folder

    mkdir webpack_resume 
    cd webpack_resume

Create src folder 
    
    mkdir src    
    
Init the yarn
    
    yarn init
    
Init the Git

    git init

Add gitignore and add the following lines

    echo '.idea*' >> .gitignore
    echo 'node_modules' >> .gitignore

Add the following libraries

- `yarn add react react-dom webpack webpack-cli`
- `yarn install`
